Text-decoration

The text-decoration property sets the apperance of decorative lines on text. It is a shorthand for "text-decoration-line", "text-decoration-color", "text-decoration-style", and the newer "text-decoration-thickness" property

Result

Text-decoration: overline;

Text-decoration: line-through;

Text-decoration: underline;

Text-decoration: underline overline;

CSS

                            
                                h1{
                                    text-decoration: overline;
                                }
                                h2{
                                    text-decoration: line-through;
                                }
                                h3{
                                    text-decoration: underline;
                                }
                                h4{
                                    text-decoration; underline overline;
                                }

Cursor

The cursor property sets the mouse cursor, if any, to show when the mouse pointer is over an element.

Result

copy

default

grab

help

no-drop

none

pointer

progress

CSS

                            
                                .copy{
                                    cursor: copy;
                                }
                                .default{
                                    cursor:default;
                                }
                                .grab{
                                    cursor:grab;
                                }
                                .help{
                                    cursor:help;
                                }
                                .no-drop{
                                    cursor: no-drop;
                                }
                                .none{
                                    cursor:none;
                                }
                                .pointer{
                                    cursor:pointer;
                                }
                                .progress{
                                    cursor:progress;
                                }

User-select

The visibility CSS property prevents text selection of a element

Result

You can select this paragraph.

You can't select this paragraph. If you double-click me, my text will not be highlighted

CSS

                            
                                .unselective{
                                    -webkit-user-select: none;(Safari)
                                    -me-user-select: none; (IE10 or IE11)
                                    user-select: none; (Standard syntax)
                                }

@keyframe
